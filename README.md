# golem-t13
Multi Layer Perceptron and Decision Tree hybrid

## Setup - uv
To install uv on lin/mac:
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
For windows:
```bash
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```
## Updating
Create a venv with uv
```bash
uv sync
```
Thats it.
